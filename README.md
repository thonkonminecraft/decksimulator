# Deck Simulator for Decked Out 2
### Fabric Minecraft mod for 1.21.1 (Requires Fabric API)

Download this mod at .

This minecraft mod is to be used with TangoTek's creation of Decked Out 2. It also supports the expanded features offered by Garibaldi's team of developers at https://modrinth.com/modpack/decked-out-2.

### How to use this mod?

By pressing default hotkey "H", you load the deck simulation UI. From here you can load in a shulker box deck of cards and run simulations of dungeon gameplay. Results are given for total loot generated, clank block and hazard block generated etc.
The settings UI provides options for run type, target level, card stats and more.
